# MASTER Discord Bot

بوت لإدارة أقسام السيرفر مع نظام أزرار تفاعلية.

## الأوامر:
- `/add_section` - إضافة قسم جديد
- `/publish` - الإعلان عن الأقسام
- `/remove_section` - حذف قسم
- `/list_sections` - عرض الأقسام
- `/update_buttons` - تحديث الأزرار

## التشغيل:
1. إنشاء ملف `.env` يحتوي على `DISCORD_TOKEN=your_token`
2. تشغيل `pip install -r requirements.txt`
3. تشغيل `python main.py`
